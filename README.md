# myRPC

myRPC — учебный клиент-серверный проект на языке C для удаленного выполнения команд через сокеты.

## Структура проекта

- `client/` — консольный клиент myRPC-client.
- `server/` — серверное приложение myRPC-server.
- `configs/` — конфигурационные файлы сервера и список разрешенных пользователей.
- `libmysyslog/` — библиотека журналирования.
- `Makefile` — общий файл сборки проекта.
