# myRPC

myRPC — клиент-серверное приложение на языке C, предназначенное для удалённого выполнения команд на сервере посредством TCP-соединения.

## Назначение проекта

Проект реализует упрощённый механизм удалённого вызова процедур (RPC). Клиент формирует запрос, содержащий имя пользователя и команду, после чего передаёт его серверу. Сервер выполняет проверку пользователя по списку разрешённых пользователей, выполняет полученную команду и возвращает результат клиенту.

## Основные возможности

* подключение клиента к серверу через TCP;
* обработка конфигурационного файла сервера;
* проверка пользователя по whitelist;
* выполнение команд на сервере;
* передача результата выполнения клиенту;
* журналирование событий;
* сохранение результата выполнения команды.

## Конфигурация сервера

Файл `configs/myRPC.conf` содержит параметры запуска сервера.

Пример содержимого:

```ini
port = 8642
socket_type = stream
```

Файл `configs/users.conf` содержит список пользователей, которым разрешено выполнение команд на сервере.

Пример содержимого:

```text
semakan
root
```

## Сборка проекта

Для сборки всех компонентов проекта необходимо выполнить команду:

```bash
make
```

Для очистки проекта:

```bash
make clean
```

## Запуск сервера

```bash
cd server
./myRPC-server
```

## Запуск клиента

```bash
cd client
./myRPC-client --host 127.0.0.1 --port 8642 --user semakan --command "whoami" --stream
```

## Алгоритм работы

1. Клиент устанавливает соединение с сервером.
2. Клиент передаёт имя пользователя и команду.
3. Сервер принимает запрос.
4. Сервер проверяет пользователя по списку разрешённых пользователей.
5. При успешной проверке выполняется команда.
6. Результат выполнения сохраняется и отправляется клиенту.
7. Клиент получает результат и выводит его на экран.
